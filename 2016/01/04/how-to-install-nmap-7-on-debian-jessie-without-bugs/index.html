How to install nmap 7 on debian jessie (without bugs)

Arnaud | 04 Jan 2016

NMAP 7.01 is out !

Nmap 7.01 is out since December 19th but my first install didn't work on debian 8 with the basic steps of the documentation :

bzip2 -cd nmap-7.01.tar.bz2 | tar xvf -
cd nmap-7.01
./configure
make
su root
make install

A bug ?

My first attempt to type zenmap in the terminal show a xfce dialog box with the following error:

"[Errno 2] File or Folder not found: '/usr/share/zenmap/config"

And the same error when typing nmap -v

If you resolve this issue, zenmap is launching but there are lot of NSE scripts errors.

It's due that you have compiled without openssl.
see nmap --versions

So how to fix it ?

sudo apt-get install libssl-dev
wget https://nmap.org/dist/nmap-7.01.tar.bz2
bzip2 -cd nmap-7.01.tar.bz2 | tar xvf -
cd nmap-7.01
./configure
make
su root
make install
mkdir -p /usr/share/zenmap/config
sudo cp /usr/local/share/zenmap/config/* /usr/share/zenmap/config/

Now we can verify in the terminal :
nmap -v nmap --version zenmap

It Works !

I think it will works on ubuntu...
